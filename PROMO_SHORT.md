# 🦞 Agent Monitor - 你的 AI Agent 全能管家

## 一句话介绍

**实时监控你的 AI Agent 状态，Token 消耗一目了然，定时任务尽在掌握！**

---

## 痛点场景

你是否也有这样的困扰？

- 🤯 多个 Agent 后台运行，不知道谁在忙？
- 💸 Token 消耗莫名其妙，想查不知道在哪看？
- ⏰ 定时任务太多，记不住什么时候执行？
- 📱 想随时随地查看状态，但没有好工具？

**Agent Monitor 就是为解决这些问题而生的！**

---

## 核心价值

### 1️⃣ 一目了然的状态面板
10个 Agent 状态实时展示，谁忙碌、谁空闲、谁有问题，一眼看清。

### 2️⃣ 精确的 Token 统计
每日消耗、累计消耗、Agent 级别明细，成本控制不再靠猜。

### 3️⃣ 完整的任务队列
定时任务状态、下次执行时间、执行结果，全部可视化。

### 4️⃣ 移动端友好
响应式设计，手机打开也能看，随时随地掌握全局。

---

## 快速体验

```bash
# 3步启动
git clone https://github.com/openclaw/agent-monitor.git
cd agent-monitor
python3 monitor.py && python3 server.py

# 打开浏览器
open http://localhost:9001
```

或者直接下载 macOS 应用，双击运行！
