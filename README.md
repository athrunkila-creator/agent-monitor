## ✨ 功能特性

### 📊 一目了然的状态面板

10 个 Agent 状态实时展示，谁忙碌、谁空闲、谁有问题，一眼看清。

| 状态 | 含义 |
|------|------|
| 🟢 活跃 | 正在工作，10分钟内有活动 |
| 🟡 忙碌 | 正在执行任务，有工具调用 |
| ⚪ 空闲 | 暂无任务，等待调度 |

### 💰 精确的 Token 统计

- **每日消耗明细**：每个 Agent 的 Token 使用量
- **累计消耗汇总**：总量一目了然
- **成本可视化**：帮你的 AI 支出做预算

### ⏰ 完整的任务队列

- 24 个定时任务状态监控
- 下次执行时间预告
- 任务执行结果追踪
- 超过 3 条任务自动折叠，点击展开

### 🎨 精美的 Dashboard

- **响应式设计**：手机、平板、电脑都能看
- **深色主题**：护眼舒适，科技感满满
- **一键刷新**：数据实时更新
- **任务排序**：主 Agent 置顶，其他按任务量排序

### 🌐 多种访问方式

- **本地访问**：`http://localhost:9001`
- **公网访问**：支持 Tailscale Funnel
- **移动端友好**：随时随地掌握全局

## 🚀 快速开始

### macOS 用户（推荐）

```bash
# 下载并解压
unzip AgentMonitor-v1.0.1-macOS.zip

# 双击运行 - 就这么简单！
open AgentMonitor.app
```

首次运行会自动：
- ✅ 检测 Python 环境
- ✅ 安装依赖包
- ✅ 初始化数据
- ✅ 打开浏览器

### 命令行启动（所有平台）

```bash
# 克隆仓库
git clone https://github.com/athrunkila-creator/agent-monitor.git
cd agent-monitor

# 安装依赖
pip3 install requests

# 启动服务
python3 monitor.py  # 初始化数据
python3 server.py   # 启动 Web 服务

# 打开浏览器
open http://localhost:9001
```
